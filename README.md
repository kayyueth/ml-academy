# ml-academy

A self-contained browser app for learning:

- machine learning basics
- federated learning fundamentals
- a standalone Numerai-style simulator workflow page

## Run

Open `/Users/k/Documents/ml-academy/index.html` in a browser.

Open `/Users/k/Documents/ml-academy/numerai.html` for the independent Numerai simulator page.

## Python Sample

A sample Python workflow is included at `/Users/k/Documents/ml-academy/numerai_training_example.py`.

Example command:

```zsh
python3 /Users/k/Documents/ml-academy/numerai_training_example.py
```

This is a teaching scaffold, not a real Numerai pipeline.
